Welcome to Module 3, Marine genomics. For this module, we worked on genome assembly for Black Band Disease, with the first three workshops acting as introductions to genomics and providing information on what we would be doing in the last workshop.

The fourth workshop is split into four parts, numbered 01 to 04 in the repository. Steps 01, 02, and 03 are merely to being the assembly process and ensure that the data is ready for analysis, which was completed in Step 4.

All of the workshops and steps are heavily annotated by me, and I have also included some of the critical data outputs from the last workshop that visualize what was done. There is also a simplified command log containing only the relevant commands and simple instructions to follow if you are more experienced with this material.

Each shell script is also self-contained within its own folder, to identify what program they were used in.
